# IMDB Clone APP
This IMDB clone app has been created using HTML, CSS, JavaScript and Bootstrap.

# Features
- Home Page:
Search bar to search any movie from the API and display the search results on the frontend. As we type the search results gets updated, just like Google does for suggestions.
Each search result of the movie has a favourite button, clicking on which the movie should gets added to “My favourite movies” list.
On clicking any particular search result (any movie), new page gets opened with more information about that movie (movie page)

- Movie Page:
This shows information about the movie like its name, photo, plot, etc.

- My favourite movies Page
Displays a list of all the favourite movies.
Remove from favourites button: Each movie has a remove from favourites button, clicking on which that movie gets removed from the list.
